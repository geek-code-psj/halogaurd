## Phase 1: Chrome Extension (Apr 13-25)

### Deliverables
- [ ] Popup UI (React component)
- [ ] Content script injection on ChatGPT/Claude/Gemini
- [ ] Real-time analysis results
- [ ] Settings panel (API endpoint, theme)
- [ ] Local testing working
- [ ] Submit to Chrome Web Store

### Architecture

```
┌─────────────────────────────────────────┐
│         Chrome Web Page (ChatGPT)       │
├─────────────────────────────────────────┤
│  Content Script (injected by extension) │
│  - Detects user input                   │
│  - Sends to popup                       │
│  - Displays overlay results             │
└──────────┬──────────────────────────────┘
           │
           │ Message API
           ↓
┌──────────────────────────┐
│   Popup Panel (React)    │
│ - Analyze button         │
│ - Show results           │
│ - Settings icon          │
└──────────┬───────────────┘
           │
           │ HTTP/WebSocket
           ↓
┌──────────────────────────────────────┐
│   HaloGuard Backend API              │
│   (halogaurd-production.up.railway.app)
│   - POST /api/v1/analyze             │
│   - WebSocket connections            │
└──────────────────────────────────────┘
```

### Key Files to Create
```
chrome-extension/
├── manifest.json              # MV3 manifest
├── public/
│   ├── icon-16.png           # Extension icons
│   ├── icon-48.png
│   ├── icon-128.png
│   └── popup.html            # Popup entry
├── src/
│   ├── popup/
│   │   ├── Popup.tsx         # Main popup component
│   │   ├── popup.css
│   │   └── index.tsx
│   ├── content/
│   │   ├── content.ts        # Inject into pages
│   │   ├── injector.ts       # Message handling
│   │   └── styles.css
│   ├── background/
│   │   └── background.ts     # Service worker
│   ├── utils/
│   │   ├── storage.ts        # Storage API wrapper
│   │   ├── api.ts            # Backend communication
│   │   └── constants.ts
│   └── types/
│       └── index.ts
├── build/          # Generated by build process
└── package.json
```

### Implementation Steps
1. Create manifest.json (MV3 format)
2. Build Popup.tsx with "Analyze This" button
3. Build content.ts to inject detection overlay
4. Setup message passing (popup ↔ content script ↔ backend)
5. Local testing on ChatGPT
6. Package for Web Store
7. Submit to Chrome Web Store
